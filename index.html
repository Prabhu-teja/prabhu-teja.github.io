Chatbot using DialogFlow
                Chatbot was developed to register the issues of the users into the Database. As the DialogFlow uses Natural Language Processing, it will be easier to understand the user queries. As the user enters his/her mobile number, Chatbot verifies the user whether he/she is registered or not. If registered, issue entered by the user will be stored into the Database and a trouble ticket ID will be generated so that the user can check the status of the issue some other time. If a user is not registerd, he/she can register by entering his/her name and mobile number.
